Vakantiewerk 2015

Je kan twee extra attributen instellen bij een form. Deze twee attributen zijn form_url en form_height. form_height is niet per se nodig, als je hem niet instelt zal deze defaulten op 700em. form_url is wel nodig. De front matter van dit form ziet er bijvoorbeeld als volgt uit: +++ date = "2015-02-03T11:38:02+01:00" draft = false url = "/vakantiewerk2015" title = "Vakantiewerk 2015" form_url = "https://docs.google.com/a/eestec.be/forms/d/1wM49GIFHhPnOR4DY7LB6HV0kq85O4G8q00I4rjJ5cx0/viewform" form_height = "620em" +++ Ik heb een testform gemaakt, met een stijl die past bij deze website.
